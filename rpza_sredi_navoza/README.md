# NeoSync Mobile

Flutter приложение для NeoSync — менеджера подписок.

## Настройка

### 1. Установи зависимости Flask
```
pip install flask-cors pyjwt
```
или
```
pip install -r requirements.txt
```

### 2. Укажи IP сервера в приложении

Открой `lib/api/api_client.dart` и замени `baseUrl`:

- **Android эмулятор**: `http://10.0.2.2:5000` (уже стоит)
- **Реальный телефон в одной WiFi**: `http://192.168.X.X:5000` (узнай IP своего ПК через `ipconfig`)
- **Продакшн сервер**: `https://your-domain.com`

### 3. Запусти Flutter
```
cd rpza_sredi_navoza
flutter pub get
flutter run
```

## Архитектура

- **БД**: единая SQLite (`instance/subscriptions.db`) — и сайт и мобилка работают с одной базой
- **Авторизация мобилки**: JWT токены через `/mobile/auth/*`
- **Авторизация сайта**: Flask-Login сессии (не меняется)
- **API мобилки**: все эндпоинты на `/mobile/*` с CORS

## Функционал

- Вход / регистрация по email+пароль
- Список подписок с карточками
- Добавление / редактирование / удаление подписок
- Детальный просмотр с кнопкой оплаты
- Аналитика: расходы в месяц/год, топ расходов
- Профиль пользователя
- Тёмная / светлая тема
